Image Processing Techniques in Autonomous Vehicle Technologies

  • Tech Stack: Python, OpenCV, Ultralytics YOLOv8, DeepSORT, NumPy, MoviePy, Google Colab
  • Github URL: Project Link

This was an academic project focused on developing an image processing pipeline for autonomous driving. The goal was to perform lane detection, vehicle detection, and multi-vehicle tracking from camera video streams by combining traditional image processing with modern deep learning models. My responsibility was to implement the vehicle detection & tracking pipeline and ensure real-time integration of lane detection with object tracking.

The system consisted of these components:

  • YOLOv8 (Ultralytics): Vehicle detection on each frame.
  • DeepSORT: ID-based multi-object tracking for consistent targets across frames.
  • OpenCV: Lane line detection and visualization (highlighting road lanes).
  • MoviePy: Generation of annotated video outputs with detections, IDs, and lanes.

The final pipeline simultaneously detects lanes, identifies vehicles, assigns unique IDs, and tracks them frame-by-frame, producing a single unified video output.

Development Environment

  • Platform: Google Colab with GPU acceleration.
  • Structure (Notebooks): serit_takip.ipynb (lane detection), car_detect_and_tracking.ipynb (vehicle detection & DeepSORT), lane_and_car_detection_tracking_pipeline.ipynb (full integration).

Datasets & Weights

  • Lane Detection: Udacity Lane Detection videos.
  • Vehicle Detection: YOLOv8 pretrained weights (COCO).
