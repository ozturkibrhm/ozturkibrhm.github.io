Medical Dataset – EDA & Preprocessing

  • Tech Stack: Python, Pandas, NumPy, SciPy, Scikit-learn, Seaborn, Matplotlib, RapidFuzz, OpenPyXL
  • Github URL: Project Link

This was an academic project focused on EDA (Exploratory Data Analysis) and preprocessing of a Physical Medicine & Rehabilitation dataset. The primary objective was to clean, standardize, and prepare the dataset for modeling, with the target variable being Treatment Duration (TedaviSuresi).

The project workflow included:

  • Data Exploration: Analyzing 13 features, studying distributions, correlations, and relationships with the target variable.
  • Missing Value Handling: Applied multiple strategies such as median imputation, mode replacement, “None” category filling, and group-based imputations.
  • Standardization: Splitting multi-valued categorical fields (e.g., allergies, chronic diseases) into normalized token lists.
  • Data Cleaning: Outlier detection using IQR, removal of anomalies, and deduplication of patient records.
  • Encoding: One-Hot Encoding (OHE) and Top-K token selection to convert categorical features into numerical form.
  • Feature Selection: Compared feature subsets for TOP-K = 20, 25, and 30; the best-performing configuration was TOP-K = 25, chosen as the final model-ready dataset.

The final deliverable was a clean and structured dataset: dataset_model_ready_best_top25.(xlsx|csv), fully ready for downstream modeling and predictive analysis.

Development Environment

  • Platform: Kaggle Notebook (Python 3.9+)
  • Libraries: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, RapidFuzz, OpenPyXL
  • Artifacts: Clean_Data_Case_DT_2025_std.xlsx, dataset_gender_filled.csv, dataset_final_imputed.csv, and final dataset_model_ready_best_top25.xlsx.

Execution Steps

  1. Load the original dataset: Talent_Academy_Case_DT_2025.xlsx.
  2. Perform EDA (distribution plots, correlation heatmaps, target relationships).
  3. Preprocess: numeric standardization, categorical encoding, and missing value imputation.
  4. Clean anomalies and remove duplicates.
  5. Apply encoding & Top-K feature selection.
  6. Export the final dataset for model training.
