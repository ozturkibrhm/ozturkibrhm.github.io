Google Drive File & Email Automation

  • Tech Stack: C#, ASP.NET Core, Google Drive API, Gmail SMTP, OAuth2
  • Github URL: Project Link

This was a personal project I developed to integrate Google Drive operations with automatic email notifications. The main goal of the project was to provide a seamless way to create and copy folders in Google Drive, manage file permissions, and share files with other users by automatically sending emails that include the share link.

The system consisted of multiple modules:

  • Google Drive Service: Provided functionalities for folder creation, duplication, and file permission management via OAuth2 integration.
  • Email Service: Automated email sending with Gmail SMTP, delivering file names and shareable links directly to recipients.
  • Collaboration Automation: Ensured that project files were instantly shared with relevant users without manual intervention.
  • Database & Config Management: Maintained secure storage of OAuth2 credentials and streamlined configuration handling.

My contributions were mainly in implementing the Google Drive Service for folder and permission management, as well as developing the Email Service that securely delivers shared file links to end users. I used Google.Apis.Drive.v3 for Drive integration and System.Net.Mail with Gmail SMTP for mail automation.
